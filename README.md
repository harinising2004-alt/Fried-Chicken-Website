# 🍗 TFC Nation - Tender Fried Chicken Website

A modern and responsive restaurant landing page built using HTML and CSS. This project showcases a fried chicken restaurant website with attractive design, product listings, services, contact information, and smooth navigation.

## 🚀 Features

* Responsive Navigation Bar
* Hero Banner Section
* Features Section
* About Us Section
* Product Showcase
* Services Section
* Contact Form
* Social Media Footer
* Smooth Scrolling Navigation
* Modern UI Design
* Custom CSS Animations

## 🛠️ Technologies Used

* HTML5
* CSS3
* Google Fonts
* SVG Icons
